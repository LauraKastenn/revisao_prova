DEFINIÇÃO:

O CSS é a sigla para Cascading Style Sheets, que significa Folha de Estilo em Cascatas.
É essa a linguagem que irá programar o visual do site, a aparência.

ESTRUTURA

Para chamar uma classe use '.' antes de escreve-la, para chamar um id use '#', já se quiser chamar
uma tag não é necessario colocar nada antes.
