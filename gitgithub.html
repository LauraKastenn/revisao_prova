Git é um VCS, que significa "Version Control System" (Sistema de Controle de Versão). O Git é um dos sistemas de controle de versão mais populares e é considerado a ferramenta padrão para desenvolvimento de software. É amplamente utilizado para gerenciar o código-fonte de projetos, permitindo que múltiplas pessoas trabalhem simultaneamente sem conflitos.

GITHUB:

O GitHub é uma plataforma baseada em nuvem em que é possível armazenar, compartilhar e trabalhar com outras pessoas para escrever códigos.
