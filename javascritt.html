JAVASCRIPT

DEFINIÇÃO:

JavaScript é uma linguagem de programação que permite
criar páginas web interativas, tornando aplicações Web
mais dinâmicas
FRONT-END: Interatividade com o cliente, uso
do DOM (Document Object Model) e o AJAX,
que carrega dados dinamicamente;
BACK-END: uso do NODE.JS

CRIAÇÃO:

O JavaScript foi criado em 1995 por Brendan
Eich, um engenheiro da Netscape. A linguagem foi
inicialmente chamada de Mocha,
depois LiveScript, e finalmente JavaScript.



ELEMENTOS

- Variáveis: Armazenam valores e podem ser declaradas com `var`, `let` ou `const`. Tipos incluem string, número, booleano, array e objeto.
- Exemplo: `let nome = "Laura";`

- Funções: Blocos de código reutilizáveis definidos com `function`, que aceitam parâmetros.
- Exemplo: `function saudacao(nome) { return "Olá, " + nome; }`

- Eventos: Ações como cliques, capturadas com `addEventListener`, que permitem interatividade.
- Exemplo: `elemento.addEventListener("click", função);`

- Arrays: Estruturas que armazenam múltiplos valores acessíveis por índices.
- Exemplo: `let frutas = ["maçã", "banana"];`

- Objetos: Estruturas complexas com propriedades e métodos, acessíveis por notação de ponto ou colchetes.
- Exemplo: `let carro = { marca: "Fusca", ano: 1970 };`

- Condicionais: Permitem decisões no código usando `if`, `else`, `else if`, `switch`, etc.
- Exemplo: `if (condição) { ... } else { ... }`

- Loops: Estruturas para repetição, como `for`, `while`, e `do-while`.
- Exemplo: `for (let i = 0; i < 5; i++) { ... }`

- Operadores: Realizam operações matemáticas e lógicas.
- Tipos: Aritméticos, Lógicos e de Comparação.
- Exemplo: `let soma = a + b;`

- Strings: Sequências de caracteres manipuláveis com métodos como `replace()`, `substring()`, etc.
- Exemplo: `let textoModificado = texto.replace("mundo", "JavaScript");`

- Escopos: Definem a visibilidade das variáveis (global e local).
- Exemplo: `let globalVariable = "Eu sou global";`

- DOM (Document Object Model)**: Representa a estrutura de uma página HTML, permitindo manipulação dinâmica.
- Exemplo: `document.getElementById("titulo").innerText = "Novo Título";`

- Eventos de Tempo: Métodos como `setTimeout` e `setInterval` executam códigos após um intervalo ou repetidamente.
- Exemplo: `setTimeout(function() { ... }, 2000);`
