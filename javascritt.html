JAVASCRIPT

DEFINIÇÃO:

JavaScript é uma linguagem de programação que permite
criar páginas web interativas, tornando aplicações Web
mais dinâmicas
FRONT-END: Interatividade com o cliente, uso
do DOM (Document Object Model) e o AJAX,
que carrega dados dinamicamente;
BACK-END: uso do NODE.JS

CRIAÇÃO:

O JavaScript foi criado em 1995 por Brendan
Eich, um engenheiro da Netscape. A linguagem foi
inicialmente chamada de Mocha,
depois LiveScript, e finalmente JavaScript.



ELEMENTOS

- Variáveis
Armazenam valores e podem ser declaradas com var, let ou const.
Tipos: var, let, const - podendo ser string(texto), numero, booleano (true or false), array e objeto
Exemplo: let nome = "Laura";

- Funções
São blocos de código reutilizáveis, criados com a palavra-chave function, que permitem a execução de instruções com parâmetros.
Definição: function nomeDaFuncao(parâmetros) { ... }
Exemplo: function saudacao(nome) { return "Olá, " + nome; }

- Eventos
Representam ações como cliques e carregamentos de página, capturados por meio de addEventListener, possibilitando a interatividade.
Método: addEventListener(evento, função)
Exemplo: elemento.addEventListener("click", função);

- Arrays
Estruturas que armazenam múltiplos valores, acessíveis por índices.
Declaração: let array = [valor1, valor2, ...]
Exemplo: let frutas = ["maçã", "banana"];
Declaração: let objeto = { propriedade1: valor1, propriedade2: valor2 }
Exemplo: let carro = { marca: "Fusca", ano: 1970 };

- Objetos
Os objetos são estruturas complexas que contêm propriedades e métodos, acessíveis por notação de ponto ou colchetes.
Declaração: let objeto = { propriedade1: valor1, propriedade2: valor2 } Exemplo: let carro = { marca: "Fusca", ano: 1970 };

- Condicionais
Permitem a tomada de decisões no código utilizando if, else e else if
Estruturas: if e else, while e do, for, switch e case. Exemplo: if (condição) { ... } else { ... }
- Loops
Tipos: for, while, do-while
Exemplo: for (let i = 0; i < 5; i++) { ... }

- Operadores
Realizam operações matemáticas e lógicas.
Tipos: Aritméticos (+, -, *, /), Lógicos (&&, ||, !) e de Comparação (==, ===, !=, !==, <,>)
1. =: Operador de atribuição 2. ==: Comparação 3. ===: Compara valor e tipo Exemplo: let soma = a + b;

- Strings
São sequências de caracteres que podem ser manipuladas de várias maneiras.
Métodos: replace(), substring(), toUpperCase(), toLowerCase()
Exemplo: let textoModificado = texto.replace("mundo", "JavaScript");

- Escopos
Definem a visibilidade das variáveis, com escopos global e local para evitar conflitos. Tipos: Global e Local
Exemplo: let globalVariable = "Eu sou global";

- DOM (Document Object Model)
O DOM representa a estrutura de uma página HTML, permitindo a manipulação dinâmica dos elementos. Métodos: getElementById(), querySelector(), innerText, appendChild()
Exemplo: document.getElementById("titulo").innerText = "Novo Título";

- Eventos de Tempo
Como setTimeout e setInterval, executam códigos após um intervalo ou repetidamente.
Métodos: setTimeout(), setInterval()
Exemplo: setTimeout(function() { ... }, 2000);
