Perguntas e Respostas

Qual é a função do comando git clone?

É usado para criar uma cópia local de um repositório remoto.
Isso permite que você trabalhe em uma versão do projeto em seu próprio ambiente.

O que faz o document.getElementById() em JavaScript?

Retorna um elemento HTML com o ID especificado,
permitindo que você interaja e manipule esse elemento no DOM (Document Object Model).

Qual é a função do display: flex; em CSS?

Ativa o modelo de layout flexível, permitindo
que os elementos filhos de um contêiner se ajustem em uma única linha ou coluna,

O que é um pull request no GitHub?

É uma solicitação para mesclar alterações feitas em um branch de um repositório em outro branch.

Como você inicia um repositório Git?

Você pode iniciar um repositório Git usando o comando 'git init' no terminal.

O que é um commit no Git?

Um commit é uma captura do estado atual do seu repositório, incluindo as alterações feitas.
