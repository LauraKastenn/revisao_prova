HTML

DEFINIÇÃO:

O HTML é uma linguagem de marcação, ou seja, ele define a
estrutura da web, permitindo a criação de páginas e a
inserção de conteúdos, incluindo imagens e vídeos.



ESTRUTURA

    - HTML: Começo
    - head: Os conteudos que ali estão não apreceram a página
    - body: Os conteudos que ali estão apreceram a página
    - header: Cabeçalho da página
    - main: Parte principal
    - footer: Rodapé
    Dica: Se escrever só o comando "Doc" ou "!DocType", a estrutura aparecerá pronta

TAGS

class: Atribui uma classe ao elemento e afeta mais de uma div;

id: Atribui um id a um unico elemento;

lang: Define o idioma principal do elemento;

titllang: Define o título do elemento;

alt: Define um texto alternativo e, muito utilizado em imagens;

hidden: Oculta o elemento;

p: Adiciona parágrafo;

text: Adiciona texto;

img: Adicionar imagem (src:enderço);

br: Quebra linha;

hr: Gera linha visual;

Th: Cabeçalho na tabela;

Tr: Linha na tabela;

a: Ancora link;

link: Conecta ao css;

main: Define o conteudo principal dentro do body;

nav: Seção com links de navegação;

section: Separar o conteúdo de uma página em blocos;

ul: Representar um item que faz parte de uma lista, nele deve

estar contido em um elemento pai: uma lista ordenada (ol) uma lista desordenada (ul) , ou um menu
